FETP India Decision Aid Tool

A comprehensive tool to design and evaluate Field Epidemiology Training Program (FETP) configurations in India, incorporating stakeholder willingness-to-pay (WTP), endorsement rates, cost-benefit analysis, Quality-Adjusted Life Years (QALYs), and probabilistic sensitivity analysis (PSA).

Welcome to the FETP India Decision Aid Tool

Purpose: This tool empowers decision-makers to evaluate Field Epidemiology Training Program (FETP) expansion options in India. It integrates discrete-choice experiment (DCE) data on stakeholder willingness-to-pay (WTP) and endorsement rates with cost estimates, health-economic outcomes (QALYs), and probabilistic sensitivity analysis (PSA) to support evidence-based planning.

How to Use: Navigate to the Inputs tab to configure program attributes such as type, duration, and delivery mode. Adjust sliders for capacity and costs, then click 'Calculate & View Results' to generate outputs across tabs. Save scenarios for comparison and export results to PDF for stakeholder reports.

  • Configure: Select attribute levels (e.g., program type, duration, focus) to design FETP scenarios.
  • Evaluate WTP: Assess marginal and total WTP based on stakeholder preferences.
  • Analyze Costs: Compute present-value costs, Benefit-Cost Ratio (BCR), and net benefits in ₹.
  • Measure Health Impact: Estimate QALYs and Incremental Cost-Effectiveness Ratio (ICER) against thresholds.
  • Assess Uncertainty: Run Monte Carlo PSA to evaluate robustness under uncertainty.
  • Save & Share: Store multiple scenarios and export detailed PDF reports.

Note: Calculations use placeholder DCE coefficients and cost assumptions. Replace with actual data for production use. Ensure JavaScript is enabled for full functionality.

Willingness-to-Pay (WTP) Analysis

Purpose: Evaluate stakeholder valuation of FETP program attributes. Marginal WTP reflects the additional value of each attribute level relative to the reference. Total WTP is calculated as: (Sum of marginal WTP for selected levels) × endorsement rate × stakeholders.

Usage: After configuring a scenario in the Inputs tab, click 'Show Chart' to visualize marginal WTP for each attribute level. Use this to understand which program features stakeholders value most.

Note: Results require a calculated scenario. If no results appear, return to the Inputs tab and click 'Calculate & View Results'.

Endorsement Rate Analysis

Purpose: Predict the percentage of stakeholders likely to support the configured FETP program based on a logit model using DCE coefficients. Higher utility from selected attributes increases endorsement probability.

Usage: After calculating a scenario, click 'Show Plot' to view the endorsement rate as a percentage. This helps gauge stakeholder acceptance of the program design.

Note: Requires a calculated scenario. Coefficients are placeholders; update with actual DCE data for accuracy.

Cost–Benefit Analysis (₹)

Purpose: Compare present-value costs (fixed, per-trainee, and MoH contributions) against monetized benefits derived from WTP. Outputs include Benefit-Cost Ratio (BCR) and net benefit to assess financial viability.

Usage: Results display automatically after calculation. Click 'Toggle Cost Breakdown' to view detailed cost components (e.g., staff, equipment). Use this to identify cost drivers and optimize program design.

Note: Requires a calculated scenario. Costs are adjusted by attribute-specific multipliers.

QALY-Based Cost-Effectiveness

Purpose: Estimate health impacts in Quality-Adjusted Life Years (QALYs) gained from trained epidemiologists. The Incremental Cost-Effectiveness Ratio (ICER) compares costs to QALYs against user-defined thresholds to assess cost-effectiveness.

Usage: Adjust QALY gain per trainee and threshold values, then click 'Run QALY Analysis' to view results and a chart comparing ICER to the threshold. Use this to evaluate health-economic value.

Note: Requires a calculated scenario. Default QALY gain is 0.03/trainee/year; adjust based on context.

Probabilistic Sensitivity Analysis (PSA)

Purpose: Assess the robustness of results under uncertainty using Monte Carlo simulations. Applies truncated normal distributions to costs, WTP, and endorsement rates, reporting means, 95% confidence intervals, and probabilities (e.g., P(BCR>1)).

Usage: Adjust standard error percentages (SE%) for costs, WTP, and endorsement, and set iteration count. Click 'Run PSA' to generate results and visualize BCR and ICER distributions.

Note: Requires a calculated scenario. Higher iterations improve accuracy but increase computation time.

Methodology

Overview: This section details the computational methods and assumptions underlying the tool’s analyses. Calculations use placeholder coefficients and should be updated with actual DCE and cost data for real-world application.

Cost Calculations

Costs are calculated as present value (PV) over the specified horizon: PV Cost = Σ_t [(Fixed costs × multipliers) + (Per-trainee costs × endorsed trainees) + (MoH training cost)] / (1 + discount rate)^t. Multipliers adjust for program type, duration, focus, delivery mode, and response capacity.

Benefits (WTP)

Total benefit = (Sum of marginal WTP for selected attribute levels) × endorsement rate × number of stakeholders. Marginal WTP is derived from DCE marginal rates of substitution, reflecting stakeholder preferences.

QALYs and ICER

PV QALYs = Σ_t [capacity × cohorts/year × endorsement rate × QALY/trainee] / (1 + discount rate)^t. ICER = PV Cost / PV QALYs, compared to thresholds (e.g., ₹300,000/QALY) to determine cost-effectiveness.

Probabilistic Sensitivity Analysis (PSA)

PSA uses Monte Carlo simulations with truncated normal distributions for costs, WTP, and endorsement (e.g., endorsement constrained to 0.01–0.99). Outputs include mean, 95% CI for BCR and ICER, and probabilities like P(BCR>1).

Assumptions: Linear utility in logit model; constant QALY gain per trainee; costs scaled by attribute-specific multipliers. Update coefficients with actual data for accuracy.
